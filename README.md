# shaoxuetang-assets

浙大少学堂公众号长期复用素材：角色 IP 表情包 + 品牌图。

## CDN URL 规则

```
https://cdn.jsdelivr.net/gh/slink711/shaoxuetang-assets@main/<目录>/<文件名>
```

例：

- 何老师·翻书 → `https://cdn.jsdelivr.net/gh/slink711/shaoxuetang-assets@main/何老师/翻书.png`
- 小马虎·灯泡顿悟 → `https://cdn.jsdelivr.net/gh/slink711/shaoxuetang-assets@main/小马虎/灯泡顿悟.png`
- 品牌图·文末 → `https://cdn.jsdelivr.net/gh/slink711/shaoxuetang-assets@main/品牌图/品牌图_文末.png`

## 目录

